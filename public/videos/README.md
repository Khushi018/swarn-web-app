# Videos Folder

Place your video files here. The Reels component expects videos in the following format:

- `reel1.mp4` - First reel video
- `reel2.mp4` - Second reel video
- `reel3.mp4` - Third reel video
- `reel4.mp4` - Fourth reel video
- `reel5.mp4` - Fifth reel video

Optional thumbnail images:
- `reel1-thumb.jpg` - Thumbnail for reel1
- `reel2-thumb.jpg` - Thumbnail for reel2
- etc.

**Video Format Recommendations:**
- Format: MP4 (H.264 codec recommended for browser compatibility)
- Aspect Ratio: 9:16 (vertical/portrait orientation)
- Resolution: 1080x1920 or similar vertical format
- Duration: 15-60 seconds (typical reel length)

**Note:** If videos are not available, the component will display a gradient background with a placeholder message.
